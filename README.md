# setup_env

Setup zsh & tmux.

## Install

Please run ```./install.sh``` in the repo.

### For normal install

```
git clone git@github.com:YuZhong-Chen/.setup_env.git ~/.setup_env
cd ~/.setup_env && ./install.sh
```

### For ros1 install

```
git clone git@github.com:YuZhong-Chen/.setup_env.git ~/.setup_env
cd ~/.setup_env && ./install.sh ros1
```
### For ros2 install

```
git clone git@github.com:YuZhong-Chen/.setup_env.git ~/.setup_env
cd ~/.setup_env && ./install.sh ros2
```
